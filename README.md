# HSHB Tech 官方网站前端

这是 HSHB Tech 的官方企业官网前端项目，使用 Vite + React 构建，并部署到 Vercel。

## 网站定位

HSHB Tech is a remote-first technology service provider focused on software development, AI workflow automation, cloud setup, and practical technical consulting.

HSHB Tech は、ソフトウェア開発、業務自動化、AI ワークフロー支援、クラウド導入を提供するリモートファーストのテクノロジーサービスです。

## 页面结构

- `/` 首页
- `/about` 关于 HSHB Tech
- `/services` 服务介绍
- `/contact` 联系我们
- `/privacy` 隐私政策
- `/terms` 服务条款
- `/security` 安全说明
